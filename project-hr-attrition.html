HR Attrition Analytics & Prediction

Binary classification of employee attrition (Termd)

This project builds a principled pipeline to analyze and predict employee attrition. It emphasizes leakage control, reproducible preprocessing, and clear model diagnostics. Two models are compared: logistic regression and random forest.

Problem & Data

  • Target: Termd (1 = terminated).
  • Parsing: dates normalized (DOB, DateofHire, DateofTermination, LastPerformanceReview_Date).
  • Leakage control: identifiers and post-event columns removed (e.g., termination date/reason, manager name).

Feature Engineering

Derived continuous features: Age, TenureYears (to termination or today), and YearsSinceLastReview. Values are sanity-checked (e.g., ranges, non-negativity). Textual performance label is dropped in favor of the numeric PerfScoreID.

Leakage-Safe Split

A time-proxy split orders records by YearsSinceLastReview (fallback to TenureYears), then takes the last 20% as test data. If no proxy is available, the pipeline falls back to a stratified random split. This mimics forward-in-time evaluation and reduces look-ahead bias.

Preprocessing & Models

  • Pipelines: numeric → median imputation + standardization; categorical → most-frequent imputation + one-hot.
  • Models: Logistic Regression (class_weight='balanced') and RandomForest (n_estimators=400, class-balanced).

Both models are wrapped in sklearn Pipelines to avoid preprocessing leakage.

Evaluation

We report ROC–AUC, accuracy, confusion matrices, and ROC curves on the test split. Figures below are generated by the script.

Feature Importance

Random Forest importances highlight the strongest drivers of attrition after preprocessing and one-hot encoding. Top-K importances are exported and visualized.

Exploratory Insights

Department-level attrition and attendance distributions offer concrete levers for HR operations and policy.

Artifacts

  • Figures: confusion matrices, ROC curves, feature importances, EDA plots

Tech Stack: Python · pandas · numpy · scikit-learn (Pipeline, OneHotEncoder, StandardScaler, LogisticRegression, RandomForest) · matplotlib
